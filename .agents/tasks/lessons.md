- Do not imply that `/skill-name` is a reliable UI slash command for Codex skills. Tell the user to name the skill in natural language, e.g. `Use ios-iphone-deploy`, because the client may reserve slash commands for its own parser.

- A skill can be invocable by typing `$skill-name` exactly even when `$` autocomplete does not suggest it. Treat exact `$name` resolution, `$` autocomplete, and `/` slash-menu visibility as separate UI behaviors.
